# emojipy 

> is simple emoji lib for python3

# Install 
>pip

```
pip3 install emojipy
```

> git
```
git clone https://github.com/mgf15/emojipy.git
cd emojipy 
python3 setup.py install 
```

# Use 

```
>>> import emojipy
>>> emojipy.emoji('fire')
'🔥'
>>> emojipy.emoji(':fire:')
'🔥'
>>> emojipy.emoji_name('🔥')
'fire'
>>> emojipy.emoji_unicode('🔥')
'1F525'
>>> emojipy.version('🔥')
'E0.6'
>>> print(emojipy.emoji('person gesturing NO: dark skin tone'))
🙅🏿
```

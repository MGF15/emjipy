# emjipy 

> is simple emoji lib for python3

# Install 
>pip

```
pip3 install emjipy
```

> git
```
git clone https://github.com/mgf15/emjipy.git
cd emjipy 
python3 setup.py install 
```

# Use 

```
>>> import emjipy
>>> emjipy.emoji('fire')
'🔥'
>>> emjipy.emoji(':fire:')
'🔥'
>>> emjipy.emoji_name('🔥')
'fire'
>>> emjipy.emoji_unicode('🔥')
'1F525'
>>> emjipy.version('🔥')
'E0.6'
>>> print(emjipy.emoji('person gesturing NO: dark skin tone'))
🙅🏿
```
